# Modular Framework Application

Учебный проект, демонстрирующий расширяемую архитектуру приложения на основе модулей и контейнера внедрения зависимостей (Dependency Injection).

## Основная идея

Приложение построено по принципу модульной архитектуры. Ядро системы не содержит логики модулей и работает только через общий контракт.

Каждый модуль:
- имеет имя
- объявляет зависимости от других модулей
- регистрирует службы в DI контейнере
- выполняет инициализацию при запуске
